•	SUMMARY OF MODULES: -

I.	MODULE 1:
       Digital literacy is the ability to manage, access, understand, integrate , communicate ,evaluate and create information safely and appropriately through digital technologies for employment, decent jobs and entrepreneurship.
       It discuss about various online tools like Cool text, Prezi, Miro, Canva, GIF(Graphic Interface format).
•	Cool text 
        refers to text designs that stand out visually and enhance the impact of a message. It uses font, shadows, color, animations. It also has different types of text like Neon text, Glitch text, 3-D text, Handwritten Fonts, Retro/Vintage text.

•	Prezi 
        Uses Canva- based approach. It is visually captivating, zoomable canvas to generates relationship between ideas, more memorable as can use various templates to customize our presentations that includes party invitations, formal meeting invitation, demise invite etc. It’s unique zoom in- zoom out feature keeps viewers engaged and enhances creative thinking.

•	Miro
          Is a whiteboard platform which contains sticky notes, shapes adding images feature to create flowcharts, diagrams easily. It includes templates, real time collaboration. This tool is ideal for brainstorming, planning and process visualization.

•	Canva 
              Is a user-friendly online tool that helps to create presentations, videos using ready-made templates. Due to its drag and drop feature it makes creation simple, fast and professional.

•	GIF
        Is a graphical interchange format which has the ability to animate pictures in a loop. It has a NO language barrier. It is used to express our emotions like happiness, dejected, excitement , nervousness. GIPHY, REDDIT,TENDOR are platforms where GIF’s are used. Ezgif.com is a site to create GIF’s.

                It deals with  awareness cyber crime and financial fraudulent .There are various types of cyber-crimes which includes phishing, online fraud, cyber terrorism, AI powdered attack.

•	Cyber crime in personal level includes threat to individual privacy and information that can easily be misused by criminals in blackmailing ,threats and misleading.
